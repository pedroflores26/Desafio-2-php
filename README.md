# 📘 README - Exercícios Básicos de PHP

## 📌 Sobre

Este projeto contém uma série de exercícios práticos em PHP com o objetivo de desenvolver a lógica de programação utilizando:

* Variáveis e operadores
* Estruturas condicionais
* Laços de repetição
* Arrays

Todos os exercícios estão reunidos em uma única página (`exercicios.php`) que exibe os resultados diretamente no navegador.

---

## 🧠 Conteúdos Trabalhados

---

### 🔹 1. Sequenciais (Variáveis e Operadores)

Operações básicas com variáveis:

* Soma de dois números
* Média de três valores
* Conversão de metros para centímetros
* Cálculo da área de um retângulo

📌 Objetivo: praticar operações matemáticas simples.

---

### 🔹 2. Condicionais (if / else)

Tomada de decisões com base em valores:

* Verificar se número é positivo, negativo ou zero
* Verificar maioridade (18+)
* Verificar aprovação de aluno
* Descobrir o maior entre dois números

📌 Objetivo: entender lógica de decisão.

---

### 🔹 3. Condicionais (switch/case)

Estruturas com múltiplas opções:

* Dia da semana (1 a 7)
* Vogal ou consoante
* Status de pedido (ex: aguardando, enviado)

📌 Objetivo: organizar múltiplas condições.

---

### 🔹 4. Laços de Repetição (for)

Execução repetitiva com controle:

* Contagem de 1 a 10
* Números pares de 1 a 20
* Tabuada de um número

📌 Objetivo: automatizar repetições.

---

### 🔹 5. Laços de Repetição (while)

Execução baseada em condição:

* Contagem regressiva
* Soma de 1 até 100

📌 Objetivo: entender repetição com condição.

---

### 🔹 6. Laço (do-while)

Execução garantida ao menos uma vez:

* Sorteio até acertar o número 5
* Contagem de tentativas

📌 Objetivo: executar primeiro, validar depois.

---

### 🔹 7. Arrays (Vetores)

Manipulação de listas de dados:

* Lista de frutas com `foreach`
* Soma de valores em array
* Array associativo (dados de aluno)

📌 Objetivo: trabalhar com múltiplos valores.

---

## 📄 Arquivo Principal

### 📌 `exercicios.php`

Arquivo responsável por:

* Executar todos os exercícios
* Exibir resultados formatados em HTML
* Organizar visualmente com CSS

---

## 🎨 Interface

A página possui:

* Layout centralizado
* Blocos separados por exercício
* Destaque para resultados
* Estilo simples e limpo

---

## 🚀 Como Executar

1. Instale um servidor local (XAMPP recomendado)
2. Coloque o arquivo na pasta:

```
htdocs/
```

3. Acesse no navegador:

```
http://localhost/exercicios.php
```
